Для запуска скрипта необходимо скачать все зависимости с помощью команды npm install. После успешной установки зависимостей перейдите в корневую папку проекта и выполните команду node server.js. По умолчанию сервер будет слушать порт 3000, однако вы можете изменить этот порт в настройках. Сервер будет доступен по ссылке http://localhost:3000/run-check.
